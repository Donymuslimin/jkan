Tentang Open Data

Apa itu API Pemilu?

Berikut ini Aplikasi yang telah dibuat dengan menggunakan api pemilu.

API Pemilu dikelola oleh Perludem namun tersedia secara gratis dengan lisensi open source. Kami mengundang teman-teman developer untuk ikut menggunakan dan berkontribusi untuk mengembangkan API Pemilu.

Apa itu API?

API adalah singkatan dari Application Programming Interface. Interface atau antarmuka adalah dimana dua sistem bertemu dan berinteraksi. Sebuah API adalah antarmuka yang digunakan oleh sebuah aplikasi agar dapat berkomunikasi dengan aplikasi lain.

API Pemilu 2014 adalah sebuah aplikasi yang menggunakan antarmukanya agar informasi dan fungsionalitas tentang pemilu 2014 di Indonesia tersedia untuk aplikasi lain.
